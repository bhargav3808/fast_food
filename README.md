# FastFood Project 🍔

A full-stack food ordering and delivery web application built with Django and SQLite/MySQL. The platform allows customers to browse food categories, add products to a cart, place orders, manage delivery addresses, and track deliveries through a user-friendly interface.

## Features

* User Registration, Login, and Authentication
* Food Categories and Product Management
* Product Search and Menu Browsing
* Shopping Cart with Quantity Management
* Secure Order Placement System
* Delivery Address Management
* Order History and Tracking
* Delivery Management Module
* PDF Invoice Generation
* Admin Dashboard for Managing Products, Categories, Orders, and Deliveries
* Responsive Design for Mobile and Desktop Devices

## Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, JavaScript, Bootstrap
* Database: SQLite / MySQL
* Authentication: Django Authentication System
* PDF Generation: ReportLab

## Project Highlights

This project simulates a real-world online food delivery platform similar to Zomato or Swiggy. It demonstrates full-stack web development concepts including database design, authentication, CRUD operations, cart management, order processing, delivery tracking, and invoice generation.

## Installation

```bash
git clone <repository-url>
cd fastfood_project

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
